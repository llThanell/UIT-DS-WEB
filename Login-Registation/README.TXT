Để chạy code thì:
Mở terminal và dẫn đường dẫn đến file Login-Registation
Cài Nodejs: Này lên mạng xem :>

Lần đầu chạy, lần sau khỏi chạy lệnh này nữa:
npm init

Cài cái này để lưu trữ khi mik có thay đổi gì ở file thì trên web sẽ cập nhật ngay:
npm i --save-dev nodemon dotenv

Cài thư viện: 
npm i express ejs brcypt express-flash express-session method-override passport passport-local


Sau khi cài đầy đủ, để chạy thì: 
npm run devStart

Truy cập trên trình duyệt:
http://localhost:3000/Login

*Lưu ý: Muốn truy cập trực tiếp từ homepage thì phải mở cái npm run devStart trước rồi mới click vô nút Login bên homepage nhaaa
